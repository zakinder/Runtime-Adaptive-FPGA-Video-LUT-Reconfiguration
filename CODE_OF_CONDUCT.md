# Code of Conduct

## Purpose

This project is intended to support professional technical collaboration around FPGA video-processing architecture, runtime LUT reconfiguration, documentation, validation, and publication preparation.

## Expected Behavior

Participants are expected to:

- Communicate respectfully and professionally.
- Focus discussion on technical content, documentation quality, and implementation accuracy.
- Provide constructive feedback with clear reasoning.
- Distinguish facts, assumptions, proposed designs, and future work.
- Respect intellectual-property, licensing, and attribution boundaries.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, personal attacks, or abusive comments.
- Spam or irrelevant promotional content.
- Misrepresentation of project status or authorship.
- Posting confidential or patent-sensitive material without permission.
- Submitting unsafe or misleading implementation guidance.

## Scope

This code of conduct applies to repository issues, pull requests, discussions, documentation edits, and related project collaboration.
