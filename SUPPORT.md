# Support

## Project Support Scope

This repository is currently a documentation and publication-preparation project for the Runtime-Adaptive FPGA Video LUT Reconfiguration architecture.

## Where to Ask Questions

Use GitHub issues for:

- Documentation corrections
- Architecture clarification
- Register-map discussion
- Validation-plan suggestions
- Diagram requests
- RTL planning questions
- Publication-package suggestions

## Before Opening an Issue

Please review:

- `README.md`
- `docs/technical-overview.md`
- `docs/faq.md`
- `docs/glossary.md`
- `docs/register-map.md`
- `docs/validation-plan.md`

## Current Limitations

This repository does not yet provide production RTL, complete simulation testbenches, or hardware deployment support. Those are planned future stages.

## Security Issues

For security-related concerns, see `SECURITY.md`.
